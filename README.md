# SEL Loan Calculator

Python/Streamlit version of the SEL loan eligibility calculator.

## Run locally

```powershell
cd "C:\Users\kobijiofor\Downloads\SEL Loan Calculator_files\sel_streamlit_app"
streamlit run app.py
```

## Deploy

Push this folder to GitHub, then deploy `app.py` on Streamlit Community Cloud.

Do not commit customer bank statements or passwords to GitHub.
